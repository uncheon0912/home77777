# GitHub API & Vercel 정적 게시판 구축 기록

## 요청 정보

- 메인 디자인: 분당 욕실 리모델링 프리미엄 랜딩 페이지
- 게시판 디자인: 분당 욕실 소식 목록 및 사이드바
- GitHub 저장소: `uncheon0912/home77777`
- 대표 이미지: `screen.png`
- 배포 방식: Vercel zero-config, GitHub Contents API

## 디자인 읽기

이 작업은 분당 지역 욕실 리모델링 업체를 위한 신뢰 중심 로컬 비즈니스 랜딩 및 편집형 게시판이며, 밝은 캔버스와 단일 그린 포인트를 사용하는 프리미엄 소비자 디자인으로 해석했다.

- DESIGN_VARIANCE: 7
- MOTION_INTENSITY: 4
- VISUAL_DENSITY: 4
- 구현: 정적 HTML + CSS + 브라우저 JavaScript

## 생성 파일

- `index.html`: 랜딩 페이지와 최신 게시글 3개 미리보기
- `news.html`: 검색, 카테고리 필터, 게시글 목록
- `news-detail.html`: 마크다운 상세 렌더링 및 관리자 수정/삭제
- `news-write.html`: 관리자 전용 작성/수정
- `admin.html`: 관리자 로그인 및 게시글 대시보드
- `db.js`: 설정 병합, 로컬 캐시, GitHub Contents API, 마크다운 렌더러
- `styles.css`: 반응형 공통 디자인
- `api/config.js`: Vercel 환경 변수 전달 함수
- `config/git_config.json`: 저장소 메타데이터와 토큰 자리표시자
- `data/posts.json`: 초기 빈 게시글 데이터
- `templates/post-template.md`: 작성 템플릿
- `_agent/skills/board-builder/SKILL.md`: 스킬 사양
- `vercel.json`: clean URL 및 trailing slash 설정
- `screen.png`: 대표 프로필 이미지

## 보안 기록

실제 GitHub 토큰은 파일, HTML, 커밋에 기록하지 않는다. Vercel에서는 `GITHUB_TOKEN`과 `ADMIN_PASSWORD` 환경 변수로 주입한다.

## 복원 메모

이 기록과 각 파일의 Git 커밋을 기준으로 이전 상태를 검토할 수 있다. 게시글 저장은 GitHub API 설정이 완료된 배포 환경에서 수행한다.
